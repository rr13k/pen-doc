## 简介
[pen](https://github.com/rr13k/pen)中文文档，采用[vitePress](https://vitepress.vuejs.org/guide/getting-started)开发


## Start

```shell
    # clone项目
    git clone https://github.com/rr13k/pen-doc.git
    # 依赖安装
    yarn
    # 启动项目
    yarn dev
```
